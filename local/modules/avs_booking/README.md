# Модуль бронирования AVS (v5.2.0)

Модуль бронирования для CMS Битрикс с интеграцией LibreBooking, ЮKassa, Битрикс24 и 1С:УНФ.

## Совместимость

- PHP >= 8.2
- Битрикс >= 20.0.0
- LibreBooking >= 3.0.3

## Установка

1. Скопировать папку `avs_booking` в `/local/modules/`
2. Установить через админку: `Настройки → Модули → AVS Booking System`
3. Настроить подключение к LibreBooking API в настройках модуля
4. Добавить компонент `avs_booking:booking.form` на страницу беседки
