# Устранение неполадок

## API LibreBooking

### Ошибка: Authentication failed. HTTP Code: 404

**Причина:** Неправильный URL API

**Решение:** Проверьте настройки модуля, URL должен быть:
https://ваш-домен.ru/booking/Web/Services

### Ошибка: Authentication failed. HTTP Code: 401

**Причина:** Неправильный логин или пароль

**Решение:** Проверьте учетные данные в настройках модуля

### Ошибка: GetReservations failed. HTTP Code: 403

**Причина:** Недостаточно прав у API-пользователя

**Решение:** Назначьте пользователю роль «Application Administrators» в LibreBooking

## Форма бронирования

### Ошибка: «Выбранное время недоступно»

**Причина:** Время уже забронировано

**Решение:** Предложите клиенту другое время

### Ошибка: «Выбранное время выходит за пределы времени работы»

**Причина:** Время окончания позже времени закрытия парка

**Решение:** Уменьшите продолжительность аренды

### Не отображаются типы аренды

**Причина:** Не заполнены цены в свойствах беседки

**Решение:** Заполните поля `PRICE_HOUR`, `PRICE`, `PRICE_NIGHT` в карточке беседки

### Не отображаются слоты для почасовой аренды

**Причина:** Неправильно настроен `LIBREBOOKING_RESOURCE_ID`

**Решение:** Проверьте соответствие ID беседки в LibreBooking

## Административная панель

### Страница настроек отображается некорректно

**Причина:** Поврежденный кэш

**Решение:** Очистите кэш: `php bitrix/modules/main/cli/clear_cache.php`

### Ошибка 404 при открытии административных страниц

**Причина:** Файлы не скопированы в `/bitrix/admin/`

**Решение:** Выполните:

```bash
cp -f local/modules/avs_booking/admin/*.php bitrix/admin/avs_booking_*.php
```

### Дашборд не обновляется

**Причина:** Кэш или ошибка синхронизации

**Решение:** Нажмите кнопку «Синхронизация» или очистите кэш

## Синхронизация

### Бронирование появилось в LibreBooking, но не в Битрикс

**Причина:** Синхронизация еще не запустилась

**Решение:** Подождите до 5 минут или нажмите кнопку синхронизации в дашборде

### CRON не работает

Проверка:

```bash
# Проверить запуск
php /путь/до/сайта/local/modules/avs_booking/cron_sync.php

# Проверить права на лог
ls -la /путь/до/сайта/upload/avs_booking_cron.log

# Проверить crontab
crontab -l
```

## ЮKassa

### Ошибка при оплате: «Payment settings not configured»

**Причина:** Не заполнены Shop ID и Secret key

**Решение:** Заполните настройки ЮKassa в модуле

### Вебхук не обрабатывается

**Причина:** Неправильный URL или IP адреса

**Решение:** Убедитесь, что URL вебхука:

```bash
https://ваш-домен.ru/local/modules/avs_booking/webhook_yookassa.php
```

### Платеж создан, но статус не обновляется

**Причина:** Проблема с вебхуком

**Решение:** Проверьте логи: /upload/avs_booking.log

# Полезные команды

### Очистка кэша

```bash
php bitrix/modules/main/cli/clear_cache.php
```

### Проверка прав на файлы

```bash
find local/modules/avs_booking -type f -exec chmod 644 {} \;
find local/modules/avs_booking -type d -exec chmod 755 {} \;
```

### Просмотр логов

```bash
tail -f /путь/до/сайта/upload/avs_booking_sync.log
tail -f /путь/до/сайта/upload/avs_booking_cron.log
tail -f /путь/до/сайта/bitrix/php_errors.log
```
