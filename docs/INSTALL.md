# Установка и настройка модуля AVS Booking

## Системные требования

| Компонент    | Минимальная версия |
| ------------ | ------------------ |
| PHP          | 7.4                |
| Битрикс      | 20.0.0             |
| MySQL        | 5.6                |
| LibreBooking | 2.8                |

### Необходимые расширения PHP

- curl
- json
- mbstring
- pdo_mysql

---

## Установка

### 1. Копирование файлов

```bash
cp -r avs_booking /путь/до/сайта/local/modules/
```

### 2. Установка через админку Битрикс

Авторизуйтесь в административной панели: `https://ваш-домен.ru/bitrix/admin/`

Перейдите: `Настройки → Настройки продукта → Модули`

Нажмите «Обновить список модулей»

Найдите «AVS Booking System» в списке

Нажмите «Установить»

## Настройка

### 3. Настройка модуля

Перейдите: `Настройки → Настройки продуктов → Настройки модулей → AVS Booking`

#### Вкладка «Основные настройки»

| Поле                          | Значение                                  |
| ----------------------------- | ----------------------------------------- |
| URL API LibreBooking          | https://ваш-домен.ru/booking/Web/Services |
| Логин API LibreBooking        | (логин от LibreBooking)                   |
| Пароль API LibreBooking       | (пароль от LibreBooking)                  |
| API ключ для внешних запросов | сгенерировать случайную строку            |

#### Вкладка «API и интеграции»

| Поле                                   | Значение                   |
| -------------------------------------- | -------------------------- |
| Shop ID (ЮKassa) - Бетонные Системы    | из личного кабинета ЮKassa |
| Secret key (ЮKassa) - Бетонные Системы | из личного кабинета ЮKassa |
| Shop ID (ЮKassa) - Парк Победы         | из личного кабинета ЮKassa |
| Secret key (ЮKassa) - Парк Победы      | из личного кабинета ЮKassa |
| Webhook Битрикс24                      | URL вебхука из Битрикс24   |

#### Вкладка «Уведомления»

| Поле                       | Значение                     |
| -------------------------- | ---------------------------- |
| Email администратора       | admin@example.com            |
| Email менеджера            | manager@example.com          |
| Telegram Bot Token         | токен от @BotFather          |
| Telegram Chat ID менеджера | ID чата (через @userinfobot) |

#### Вкладка «Тарифы и цены»

| Поле                                              | Значение |
| ------------------------------------------------- | -------- |
| Начало летнего периода                            | 01.06    |
| Конец летнего периода                             | 31.08    |
| Час окончания работы летом                        | 23       |
| Час окончания работы зимой                        | 22       |
| Сумма аванса по умолчанию                         | 2000     |
| ID беседок с повышенным авансом                   | 5,6      |
| Сумма повышенного аванса                          | 5000     |
| Минимальное количество часов аренды               | 4        |
| Ограничение в выходные дни                        | no       |
| Модификатор цены в выходные                       | 1.2      |
| Праздничные даты 2026-01-01,2026-05-01,2026-05-09 |

### 4. Настройка инфоблока «Беседки»

Создайте или проверьте инфоблок с кодом besedki (ID=12) и свойствами:

| Код                      | Название                  | Тип   |
| ------------------------ | ------------------------- | ----- |
| LIBREBOOKING_RESOURCE_ID | ID ресурса в LibreBooking | Число |
| PRICE_HOUR               | Цена за час (руб)         | Число |
| PRICE                    | Цена за день (руб)        | Число |
| PRICE_NIGHT              | Цена за ночь (руб)        | Число |
| DEPOSIT_AMOUNT           | Сумма предоплаты (руб)    | Число |

### 5. Добавление компонента на сайт

В шаблон детальной страницы беседки (detail.php) добавьте:

```php
<?php
if (\Bitrix\Main\Loader::includeModule('avs_booking')) {
    $APPLICATION->IncludeComponent(
        'avs_booking:booking.form',
        '.default',
        [
            'ELEMENT_ID' => $arResult['ID'],
            'SUCCESS_PAGE' => '/booking-success/',
            'CACHE_TIME' => 3600
        ],
        null,
        ['HIDE_ICONS' => 'Y']
    );
}
?>
```

### 6. Создание страницы успеха

Создайте страницу /booking-success/ и разместите на ней код для отображения информации об оплате.

### 7. Настройка CRON для синхронизации

```bash
crontab -e
```

Добавьте строку:

```text
_/5 _ \* \* \* php /путь/до/сайта/local/modules/avs_booking/cron_sync.php >> /путь/до/сайта/upload/avs_booking_cron.log 2>&1
```

## Проверка работы

Откройте страницу любой беседки

Должна отображаться форма бронирования

Выберите дату — должны появиться доступные типы аренды

Заполните данные и нажмите «Забронировать»

## Устранение неполадок

| Модуль не виден в списке | Проверьте путь: /local/modules/avs_booking/install/index.php |
| | Проверьте права на файлы (644 для файлов, 755 для директорий) |
| Ошибка подключения к API | Проверьте настройки API в модуле |
| | Проверьте доступность LibreBooking: https://ваш-домен.ru/booking/Web/Services/ |
| Ошибка 500 при создании бронирования | Проверьте логи PHP: /bitrix/php_errors.log |
| | Проверьте, что все поля в форме заполнены |
|Синхронизация не работает | Проверьте CRON задачу: crontab -l |
| | Проверьте логи: /upload/avs_booking_cron.log |
