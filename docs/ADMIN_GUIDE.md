# Руководство для операторов колл-центра и менеджеров по продажам

## Вход в систему

1. Откройте браузер и перейдите по адресу: `https://домен.ру/bitrix/admin/`
2. Введите свои учетные данные (логин и пароль)
3. В левом меню найдите раздел **«Сервисы»** → **«AVS Booking»**

## Доступные разделы

### 1. Дашборд (главная страница)

**Путь:** Сервисы → AVS Booking → Дашборд

**Что видно:**

- Статистика бронирований за неделю и месяц
- Суммы бронирований и оплат
- Список бронирований на сегодня
- Статус беседок (свободна/занята)

**Действия:**

- Кнопка **«Синхронизация с LibreBooking»** — принудительная синхронизация бронирований

---

### 2. Управление бронированиями

**Путь:** Сервисы → AVS Booking → Бронирования

#### Просмотр списка

- Все бронирования отображаются в таблице
- Можно фильтровать по статусу и ID беседки
- Сортировка по дате создания (новые сверху)

#### Статусы бронирований

| Статус         | Цвет         | Описание                | Действия оператора |
| -------------- | ------------ | ----------------------- | ------------------ |
| Ожидает оплаты | 🟠 Оранжевый | Клиент не оплатил аванс | —                  |
| Оплачено       | 🟢 Зеленый   | Аванс оплачен           | Подтвердить        |
| Подтверждено   | 🔵 Синий     | Менеджер подтвердил     | —                  |
| Активно        | 🔵 Синий     | Бронирование в процессе | —                  |
| Завершено      | ⬜ Серый     | Услуга оказана          | —                  |
| Отменено       | 🔴 Красный   | Бронирование отменено   | —                  |

#### Действия с бронированием

**Для оплаченных бронирований:**

- ✅ **Подтвердить** — подтверждение бронирования менеджером
- 📅 **Продлить** — увеличение времени аренды (по согласованию с клиентом)
- ❌ **Отменить** — отмена бронирования

**Для подтвержденных/активных:**

- 📅 **Продлить** — увеличение времени аренды
- ❌ **Отменить** — отмена бронирования

**Как продлить время:**

1. Нажмите кнопку **«Продлить»** рядом с бронированием
2. Выберите новое время окончания в календаре
3. Нажмите **«Продлить»** (система автоматически пересчитает стоимость)

#### Фильтрация

| Фильтр     | Как использовать                      |
| ---------- | ------------------------------------- |
| По статусу | Выберите статус из выпадающего списка |
| По беседке | Введите ID беседки в поле фильтра     |
| Применить  | Нажмите кнопку **«Фильтр»**           |

---

### 3. Особые даты

**Путь:** Сервисы → AVS Booking → Особые даты

**Назначение:** Установка ограничений для конкретных дат (праздники, спецпредложения)

#### Добавление ограничения

1. Выберите **беседку** из списка
2. Выберите **дату** в календаре
3. Выберите **тип ограничения**:
   - **Только полный день** — нельзя забронировать почасово или ночь
   - **Без почасовой** — запрещена только почасовая аренда
   - **Свои типы** — выберите разрешенные типы вручную
4. При необходимости укажите **модификатор цены** (например, 1.5 = +50%)
5. Добавьте **описание** (например, «Новогодние праздники»)
6. Нажмите **«Сохранить»**

#### Удаление ограничения

- Нажмите **«Удалить»** в строке нужной даты

---

### 4. Скидки и промокоды

**Путь:** Сервисы → AVS Booking → Скидки и промокоды

#### Добавление промокода

| Поле                       | Описание                               | Пример             |
| -------------------------- | -------------------------------------- | ------------------ |
| Код промокода              | Уникальный код (заглавные буквы)       | `SPRING2026`       |
| Название                   | Описание для внутреннего использования | «Весенняя акция»   |
| Тип скидки                 | Процент или фиксированная сумма        | Процент            |
| Значение скидки            | 10 (для 10%) или 500 (для 500 руб)     | `10`               |
| Действителен с             | Дата начала действия                   | `2026-05-01 00:00` |
| Действителен до            | Дата окончания действия                | `2026-05-31 23:59` |
| Мин. сумма заказа          | Минимальная сумма для применения       | `3000`             |
| Макс. кол-во использований | Лимит применений                       | `100`              |
| Активен                    | Да/Нет                                 | Да                 |

#### Как клиент использует промокод

- Клиент вводит код в поле **«Промокод»** в форме бронирования
- Скидка применяется автоматически при расчете цены

## Часто задаваемые вопросы (FAQ)

### Вопрос: Как найти бронирование клиента, если он звонит?

**Ответ:** Откройте раздел **«Бронирования»**, найдите клиента по имени или телефону в таблице. Можно использовать поиск в браузере (Ctrl+F).

### Вопрос: Клиент хочет продлить время аренды. Что делать?

**Ответ:** В разделе **«Бронирования»** найдите заказ клиента. Если статус «Оплачено» или «Подтверждено», нажмите **«Продлить»**, выберите новое время окончания и подтвердите.

### Вопрос: Клиент хочет перенести бронирование на другой день.

**Ответ:** Самый простой способ — отменить текущее бронирование и создать новое на нужную дату. Либо используйте API для изменения времени (требует технических знаний).

### Вопрос: Почему бронирование не появляется в списке?

**Ответ:** Нажмите кнопку **«Синхронизация с LibreBooking»** на дашборде. Если не помогло — проверьте статус API в дашборде.

### Вопрос: Клиент утверждает, что оплатил, но статус «Ожидает оплаты».

**Ответ:** Проверьте в разделе **«Бронирования»** статус оплаты. Если оплата прошла, измените статус вручную на **«Оплачено»** через выпадающий список.

### Вопрос: Как отменить бронирование по просьбе клиента?

**Ответ:** В разделе **«Бронирования»** найдите заказ и выберите статус **«Отменено»** в выпадающем списке. Система автоматически отменит бронь в LibreBooking.
