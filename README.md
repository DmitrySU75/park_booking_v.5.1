# Модуль бронирования AVS

Модуль бронирования для CMS Битрикс с интеграцией LibreBooking, ЮKassa, Битрикс24 и 1С:УНФ.

## Возможности

- ✅ Онлайн-бронирование беседок (почасовая, дневная, ночная аренда)
- ✅ Проверка доступности через LibreBooking API
- ✅ Предоплата через ЮKassa
- ✅ Двусторонняя интеграция с 1С:УНФ
- ✅ Уведомления администратора по email и в Битрикс24
- ✅ Фильтрация каталога по дате
- ✅ Ценовые периоды (гибкое ценообразование)
- ✅ Сезонная настройка времени работы

## Требования

- PHP >= 7.4
- Битрикс >= 20.0.0
- LibreBooking >= 2.8
- MySQL >= 5.6

## Быстрая установка

```bash
# 1. Копирование файлов
cp -r avs_booking /путь/до/сайта/local/modules/

# 2. Установка через админку Битрикс
# Настройки → Модули → avs_booking → Установить

# 3. Настройка модуля
# Настройки → Настройки модулей → AVS Booking
```
